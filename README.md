# auth-todo-api

## Project setup

```
npm install
```

### Running Node Scripts

```
npm run start
```

### Docker commands

```
docker build -t cemalcanisci/app-todo-api .

docker run -p 127.0.0.1:3000:3000 -d --name todo-api  muratersin/todo-api
```

### Environment Variables

```
JWT_SECRET=SECRET
```
